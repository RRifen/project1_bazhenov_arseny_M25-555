# Лабиринт сокровищ

Текстовая игра-приключение, в которой игрок исследует лабиринт, собирает предметы, решает загадки и пытается найти сокровище.

## Описание

Игра представляет собой интерактивный текстовый лабиринт, где игрок:
- Перемещается по различным комнатам
- Собирает предметы и использует их
- Решает загадки и головоломки
- Ищет ключи для доступа к комнате с сокровищами
- Сталкивается с ловушками и случайными событиями

Цель игры — найти и открыть сундук с сокровищами.

## Требования

- Python >= 3.14
- Poetry (для управления зависимостями)

## Установка

1. Клонируйте репозиторий или перейдите в директорию проекта:
```bash
cd project1_bazhenov_arseny_M25-555
```

2. Установите зависимости с помощью Poetry:
```bash
poetry install
```

Или используйте Makefile:
```bash
make install
```

## Запуск

### Способ 1: Через Poetry
```bash
poetry run project
```

### Способ 2: Через Makefile
```bash
make project
```

## Команды игры

- `look` — осмотреть текущую комнату
- `go <направление>` или `north`/`south`/`east`/`west` — переместиться в указанном направлении
- `take <предмет>` — поднять предмет
- `use <предмет>` — использовать предмет из инвентаря
- `inventory` — показать инвентарь
- `solve` — попытаться решить загадку в комнате
- `help` — показать справку по командам
- `quit` или `exit` — выйти из игры
